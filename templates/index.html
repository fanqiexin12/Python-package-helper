📖 使用说明

  1. 在联网电脑上：选择你的目标系统和Python版本，添加需要下载的包，点击"直接下载包"获取.whl文件
  2. 复制命令：也可以点击"生成命令"获取pip下载命令，在联网电脑上执行
  3. 传输文件：将下载的.whl文件或包含所有包的文件夹复制到U盘中
  4. 离线电脑上：将.whl文件放到一个文件夹（如 wheels），然后运行安装命令
  5. 安装命令：使用 pip install --no-index --find-links=./wheels 包名 进行离线安装
💡 小提示：
  • 建议将要安装的所有包放在同一个文件夹中，方便管理
  • 如果某个包依赖其他包，需要同时下载所有依赖包
  • 使用 pip download 包名 --dest ./wheels 可以直接下载到指定文件夹
