📖 使用说明

  1. 在联网电脑上：选择你的目标系统和Python版本，添加需要下载的包
  2. 获取下载链接：点击"生成下载链接"，会显示每个包在PyPI的下载页面
  3. 下载包文件：访问链接下载对应平台的 .whl 文件
  4. 传输文件：将所有 .whl 文件复制到U盘中
  5. 离线电脑上：将 .whl 文件放到一个文件夹（如 wheels）
  6. 安装命令：使用 pip install --no-index --find-links=./wheels 包名1 包名2 ... 进行离线安装
💡 小提示：
  • 建议将要安装的所有包放在同一个文件夹中，方便管理
  • 如果某个包依赖其他包（如 numpy 依赖 scipy），需要同时下载所有依赖包
  • 使用 pip download 包名 --dest ./wheels 可以直接下载到指定文件夹
  • 可以一次下载所有包：pip download 包1 包2 包3 --dest ./wheels
